# GDedit
GDedit is a project that creates an external Geometry Dash editor with tweaks and fixes to the original. It is currently not very developed, but will be a usable version of the editor soon enough hopefully.
# Building / Compiling
To compile new changes, run ```bash compile.sh```, which will compile for both Windows and Linux. For Windows users, you can use WSL to compile. Building is also simple, just run ```bash compress.sh``` which will copy all new includes, libraries, and assets over to both versions, and compres them. Windows goes to .zip and Linux to .tar.gz.
