# GDedit
GDedit is a project that creates an external Geometry Dash editor with tweaks and fixes to the original. It is currently not very developed, but will be a usable version of the editor soon enough hopefully.
## Compiling
### Linux
To compile new changes, run ```bash compile.sh```. which will compile and run the app.
